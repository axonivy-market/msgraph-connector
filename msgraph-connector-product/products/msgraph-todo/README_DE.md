# Microsoft 365 ToDo Connector

Der [Microsoft Mail ToDo
API](https://docs.microsoft.com/en-us/graph/todo-concept-overview)-Konnektor von
Axon Ivy hilft Ihnen dabei, Microsoft To Do-Funktionen in Ihre Anwendung zu
integrieren. Microsoft ToDo bietet eine einfache Möglichkeit, Aufgaben zu
verwalten und Ihren Tag zu planen.

Dieser Konnektor:

- ist Teil der Microsoft Graph REST API.
- Ermöglicht eine einfache Aufgabensynchronisierung über alle Microsoft
  365-Produkte hinweg.
- Enthält eine Demo-Implementierung.
